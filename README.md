# VARUNM-NOVINTIX-all-four
Project 1: Healthcare Dataset Analysis and Prediction

Tasks Completed:
Task 1: Exploratory Data Analysis (EDA)
Task 2: Supervised Learning for predicting Test Results
Task 3: Unsupervised Anomaly Detection on Billing Amount
Task 4: AI Doctor Recommendation Generator

Dataset:
Healthcare dataset containing patient details, billing, medical conditions, medication, and test results.

Techniques Used:
Data cleaning and preprocessing
Visualizations using Matplotlib and Seaborn
Machine learning models for prediction
Isolation Forest for anomaly detection
Simple rule-based AI text generation

Outputs:
Distribution plots, frequency charts
Prediction model performance
Anomaly-marked records
AI-generated medical recommendation

Project 2: Household Electric Power Consumption Analysis

Tasks Completed:
Task 1: Exploratory Data Analysis of power usage trends
Task 2: Time-Series Forecasting for Global Active Power
Task 3: Unsupervised Clustering of daily power usage patterns
Task 4: AI Rule-Based Consumption Category Generator

Dataset:
Household electric power consumption dataset containing date, time, voltage, current, and sub-metering values.

Techniques Used:
Time-series preprocessing and merging Date-Time
Windowed dataset creation for forecasting
Machine learning models for next-hour prediction
Clustering using KMeans
Simple rule-based category assignment

Outputs:
Time-series plots and usage trends
Forecast vs actual comparison plot
Cluster visualizations
Automated usage category and suggestion
